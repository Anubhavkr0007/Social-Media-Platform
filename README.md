# Hiveup

Hiveup is a purple-themed content sharing platform with login/signup authorization. It includes a simple backend API and a front-end web app where users can create accounts, log in, and share posts.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open http://localhost:4000 in your browser.

## Features

- Sign up with username, email, and password
- Login and token-based authorization
- Browse feed of shared posts
- Create and share new posts
- Purple brand identity with Hiveup logo styling
